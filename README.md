# Tsuen Wan Government Secondary School - 65th Gala Dinner

荃灣官立中學六十五周年校慶晚宴嘉賓報到及座枱查詢系統

## 功能 Features

### 嘉賓報到 Guest Check-in (必填)
- **中文姓名** - 只能輸入中文字
- **中五/畢業班** - 4位數字年份
- **電話號碼** - 852 + 8位數字

私隱權聲明：所有收集的個人資料只用作登記入座及校友會日後聯絡之用

### 查詢座枱 Table Search
- **輸入姓名** - 可輸入中文姓名搜尋
- **輸入年份** - 可輸入畢業年份搜尋
- **即時顯示** - 輸入後即時顯示結果
- **枱號顯示** - 顯示該賓客的枱號
- **須先簽到** - 必須先填寫報到表單才能使用搜尋功能

### 場地枱位圖 Seating Plan
- **提供各檯的Location圖供參考** - 顯示場地佈局圖
- **枱號提示** - 搜尋結果顯示枱號提示

### 嘉賓名單 Guest List
- **提供全份出席者名單供參考** - 可查看所有嘉賓名單
- **顯示/隱藏** - 按鈕切換顯示

### 已簽到 Check-in Status
- **已簽到提示** - 提交後顯示「已簽到」
- **重新簽到** - 可修改資料

## 使用方法 How to Use

1. 開啟 `index.html`
2. **填寫嘉賓報到表** - 輸入中文姓名、中五/畢業班、電話號碼
3. **提交** - 點擊「提交」按鈕
4. **查詢座枱** - 輸入姓名或畢業年份
5. 查看結果中的枱號
6. 可點擊「提供各檯的Location圖供參考」參考位置

## 資料庫 Database

嘉賓資料庫包含 **981位嘉賓**，分為 **79枱**：
- 1-2號枱：貴賓席 (VIP)
- 3-8號枱：教職員/嘉賓老師
- 9-79號枱：校友 (1966-2024年畢業)
