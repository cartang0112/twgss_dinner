# Tsuen Wan Government Secondary School - 65th Anniversary Gala Dinner

荃灣官立中學六十五周年校慶晚宴貴賓報到及座枱查詢系統

## 網址 Website

https://cartang0112.github.io/twgss_dinner/

## 功能 Features

### 貴賓報到 Guest Check-in (必填)

**身份選擇 (必填)**
- 貴賓席
- 嘉賓老師
- 家長教師會
- 荃官教職員
- 校友

**校友 required fields:**
- **中文姓名(繁體)** - 只能輸入中文字
- **中五/DSE畢業班** - 4位數字年份 (1950-今年，系統會驗證)
- **電話號碼** - 852 + 8位數字 (必須8位，否則須重新輸入)

**非校友 required fields:**
- **中文姓名(繁體)** - 只能輸入中文字

**確認對話框** - 提交前確認資料正確
**即時顯示枱號** - 提交後立即顯示座枱編號

私隱權聲明：所有收集的個人資料只用作登記入座及校友會日後聯絡之用

### 查詢座枱 Table Search
- **須先簽到** - 必須先填寫報到表單才能使用搜尋功能
- **輸入姓名** - 可輸入中文姓名搜尋
- **輸入年份** - 可輸入畢業年份搜尋
- **即時顯示** - 輸入後即時顯示結果
- **枱號顯示** - 顯示該賓客的枱號

### 場地枱位圖 Seating Plan
- **提供各枱位置圖及座位編排表** - 顯示場地佈局圖
- **枱號提示** - 搜尋結果顯示枱號提示

### 嘉賓名單 Guest List
- **提供全份出席者名單供參考** - 可查看所有嘉賓名單
- **枱號排序** - 由小至大排列
- **顯示/隱藏** - 按鈕切換顯示

### 下載資料 Downloads
- **1966-1984 屆校友座位表** - 1966-1984年畢業校友座位安排
- **1985-2024 屆校友座位表** - 1985-2024年畢業校友座位安排
- **宴會餐枱佈置圖** - 場地佈置圖
- **座位編排表** - 完整座位編排表
- **顯示位置** - 簽到成功後顯示於成功訊息下方

### 已簽到 Check-in Status
- **已簽到提示** - 提交後顯示枱號
- **重新簽到** - 可修改資料

## 使用方法 How to Use

1. 開啟網站或 `index.html`
2. **選擇身份** - 貴賓席/嘉賓老師/家長教師會/荃官教職員/校友
3. **填寫資料** - 根據身份填寫相應欄位
4. **確認提交** - 確認資料無誤後按「確定」提交
5. **查看枱號** - 提交後即時顯示枱號
6. **查詢座枱** - 輸入姓名或畢業年份搜尋

## 資料庫 Database

嘉賓資料庫包含 **1000+位嘉賓**，分為 **81枱**：
- 1-2號枱：貴賓席 (VIP)
- 3-8號枱：教職員/嘉賓老師/家長教師會
- 9-81號枱：校友 (1966-2024年畢業)
