# Tsuen Wan Government Secondary School - 65th Gala Dinner

荃灣官立中學六十五周年校慶晚宴嘉賓報到及座枱查詢系統

## 功能 Features

### 嘉賓報到 Guest Check-in
- **中文名** (必填) - 只能輸入中文字
- **英文名** (選填) - 只能輸入英文字母
- **畢業年份** (必填) - 4位數字
- **手提電話** (必填) - 852 + 8位數字
- **電郵** (選填) - 電郵格式驗證

### 查詢座枱 Table Search
- **輸入姓名** - 可輸入中文姓名搜尋
- **輸入年份** - 可輸入畢業年份搜尋
- **即時顯示** - 輸入後即時顯示結果
- **枱號顯示** - 顯示該賓客的枱號

### 場地枱位圖 Seating Plan
- **查看枱位圖** - 顯示場地佈局圖
- **枱號提示** - 搜尋結果顯示枱號提示

### 嘉賓名單 Guest List
- **完整名單** - 可查看所有嘉賓名單 (981人，79枱)
- **顯示/隱藏** - 按鈕切換顯示

## 使用方法 How to Use

1. 開啟 `index.html`
2. **嘉賓報到** - 填寫資料並提交
3. **查詢座枱** - 輸入姓名或畢業年份
4. 查看結果中的枱號
5. 可點擊「查看場地枱位圖」參考位置

## 資料庫 Database

嘉賓資料庫包含 **981位嘉賓**，分為 **79枱**：
- 1-2 號枱：貴賓席 (VIP)
- 3-8 號枱：教職員/嘉賓老師
- 9-79 號枱：校友 (1966-2024年畢業)

資料庫儲存在 `index.html` 的 JavaScript 中，编辑 `database` 陣列可修改資料：

```javascript
const database = [
    { name: "姓名", year: 2024, table: 1 },
    // 在此新增更多嘉賓
];
```
